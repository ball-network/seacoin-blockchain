

海洋污染是一個國際性的問題。 據不完全統計，世界上每年把1000萬噸石油排入海洋； 約有100萬噸DDT已滯留在海水中。 保護海洋環境，防止海洋污染是各國的共同要求。 海洋污染的特點是污染源廣，有毒有害物質的種類多，擴散範圍大，危害深遠，控制複雜，治理難度大。 囙此，海洋污染比起陸上的其它環境污染要嚴重和複雜。
